# Proyecto Amor 💖

Este proyecto es una página web interactiva con recuerdos, canciones y juegos.  
Creado con HTML, CSS y JavaScript, diseñado con cariño para representar una historia especial.

## Estructura
- 🎵 Canciones personalizadas
- 📸 Galería de fotos
- 📝 Mensajes especiales
- 🎮 Juego de adivinanza
